## aleph-comandos alpha-2.0.0

Este paquete es una rama del paquete estable que recopilación varios comandos de uso común entre los colegas de Andrés Merino, dentro de su proyecto personal Alephsub0 (https://www.alephsub0.org/).  Provee multiples comandos para facilitar la escritura de textos con contenidos matemáticos. En esta rama se implementan nuevas características en pruebas que posteriormente pueden ser implementadas en la versión estable del software.

### Últimos cambios

```
alpha-2.0.0  2022-06-29
    - Se incluye una selección de comandos para la escritura de conceptos de Topología, Teoría de la Medida y Teoría de Probabilidades
```
